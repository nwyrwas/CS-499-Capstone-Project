📘 Project Overview

The Animal Shelter Data Analytics Dashboard represents the culmination of my Bachelor’s program in Computer Science at Southern New Hampshire University. It unifies the disciplines of software engineering, secure coding, data management, and visualization into a real-world, production-ready system. The dashboard enables shelters and organizations to make informed decisions by visualizing adoption trends, rescue rates, and location-based insights.

Built using Python’s Dash and Flask frameworks, with MongoDB as the back-end and Matplotlib for graphical analysis, the system demonstrates full-stack integration. The project evolved from a static CRUD application into an advanced analytics platform, applying modular software design and security-first principles. The interface is designed for ease of use, responsiveness, and accessibility, providing meaningful data exploration tools in an intuitive format.

🎯 Objectives

The primary objective of this capstone was to transform the static CS-340 CRUD system into an interactive, secure, and analytical platform. The goal was to provide meaningful, data-driven insights through modern visualization techniques while maintaining strict software quality standards.

Objectives included strengthening input validation, improving the user interface, optimizing MongoDB query performance, and implementing modular, maintainable architecture. These objectives were achieved by using Dash’s reactive components, Flask routing, and security policies aligned with OWASP and SEI CERT standards.

🧩 Original Artifact – CS-340 Client-Server Development

The original CS-340 artifact was a Flask-based web application that supported basic CRUD operations on a MongoDB database. It provided the foundation for database integration but lacked analytical tools and strong security design. The interface was basic, and user input validation was minimal, which presented opportunities for improvement.

By enhancing this system, I demonstrated how an existing project can evolve into a robust, production-quality product through iterative improvement, code refactoring, and adherence to secure design principles.

🚀 Enhanced Artifact – Capstone Dashboard

The enhanced version features an integrated data visualization dashboard built using Dash, Flask, and Matplotlib. Users can search, filter, and visualize adoption trends interactively. The MongoDB backend supports parameterized queries, optimized indexes, and aggregation pipelines for efficient performance.

The user interface was redesigned with a dark theme for better readability, accessibility, and professional presentation. These enhancements collectively modernized the application into an analytics platform suitable for deployment and further expansion.
